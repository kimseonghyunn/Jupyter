리스트

여러 정보를 한번에 표현
어떠한 자료형도 포함 가능

```python
odd = [1, 3, 5, 7, 9]
odd
```




    [1, 3, 5, 7, 9]


리스트 인덱싱

문자열과 유사한 방법으로 인덱싱

```python
a=[1,2,3]
```


```python
a[0]
```




    1




```python
a[1]
```




    2




```python
a[-1]
```




    3




```python
a[0]+a[2]
```




    4


리스트 안의 리스트

내부 리스트의 요소에도 인덱싱 가능

```python
a = [1,2,3,['a','b','c']]
```


```python
a[3][0]
```




    'a'




```python
a[3][2]
```




    'c'




```python
a[3]
```




    ['a', 'b', 'c']


리스트 슬라이싱

문자열과 유사한 방법으로 슬라이싱

```python
a = [1,2,3,4,5]
a[0:2]
```




    [1, 2]


중첩 리스트 슬라이싱

```python
a = [1,2,3,['a','b','c'],4,5]
a[2:5]
```




    [3, ['a', 'b', 'c'], 4]




```python
a[3][:2]
```




    ['a', 'b']


quiz

a = [1,2,3,4,1,2,3,4] 리스트에서 슬라이싱을 활용하여 [3,4,1] 리스트를 만들어보세요.

a=[1,2,3,4,1,2,3,4]
